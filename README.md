A space style modification for Rusted Warfare
Only support Chinese in game now

The mod is still on devlopment,if you see some error happen,plz propose a issue.
## How to start(in game)
The mod have 5 resource
### sulfide
explosive powder,useful for landmine
### red mine
Main resources of this mod.In base' UI can convert from credits($) as start resources
### data
Unlike normal resoures,data have a limit'capacity'.
When data reached capacity limit,it will stop increase.
But if data significantly exceeding capacity,it won't become less
